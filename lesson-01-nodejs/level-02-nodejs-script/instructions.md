# NodeJS Script

## Objective

Write a simple script and run it with NodeJS.

## Benefits

You will get familiar with running code in the terminal, where backend programming is done.

## Complete these tasks

1. In `script.js`, use `console.log` to display a message.
2. Let the message explain how to run JavaScript files in the terminal.
3. In the terminal, navigate to the script's folder.
4. Run the script in the terminal.

## More Information

- NodeJS is terminal based.
- In full stack web development, backend programming heavily involves the terminal.

## Usage Tips

- The `.js` can be omitted when running JavaScript files with `node`.

## Hints

- Use the `cd` command to navigate to specific folders.

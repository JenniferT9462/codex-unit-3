# Array Traversal

## Objective

Use NodeJS to traverse an array.

## Benefits

Practice to get better with loops.

## Complete these tasks

1. Write the code from scratch. Do not copy/paste.
2. In `script.js`, create an array with 10 values.
3. Add a `for` loop with a starting index and an ending condition.
4. Use the loop to traverse the array.
5. Display each value in the array, but number them from 1 to 10.
6. Run the script.

## More Information

- The `for` loop has a syntax that take the starting number, ending condition, and incrementor.

## Usage Tips

- A template string can display a variable. Example: \``My value is ${variable}.`\`
- Concatenate syntax can display variables. Example: `"My value is " + variable + "."`

## Hints

- Use a template string or concatenate syntax.
- Use the array index number to calculate the item number.
